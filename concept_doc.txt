		Общее описание
Данный сайт представляет собой веб-приложение для проведения расчета осевого компрессора
и извлечения результатов в графической и табличной форме.

Логика работы:
	Модуль задания исходных параметров
	Параметры, определяемые пользователем, делятся на 2 группы. 
	Первая группа: параметры, которые
просто вводятся в текстовое поле и в дальнейшем в нем же редактируются. К этим параметрам
относятся расход, необходимая степень повышения давления, ожидаемый КПД...
	Вторая группа: параметры, которые задают распределение некоторых величин 
(коэффициент расхода, коэффициента теоретического напора...). Эти параметры могут вводиться как
в текстовом поле, так и редактироваться с помощью бегунков.
	На этапе задания исходных параметров пользователь задает имя своему набору исходных данных,
которое может использоваться в дальнейшем при анализе результатов расчета. 
	При этом наборы граничных условий делятся на два вида: 
		1. Расчет единичного компрессора.
		2. Расчет, диапазона компрессоров.

После подтверждения ввода данных и удачного расчета производится редирект на страницу
выбора модуля обработки данных: всего существует три модуля анализа решения:
	1. Аналитический модуль (доступен только для диапазона компрессоров) - позволяет
		строить сравнительные аналитические графики для выборки компрессоров, создавать и
		сохранять подвыборки и, в конце концов, производить выбор единичного компрессора.
		
	2. Геометрический модуль (доступен только для единичного спрофилированного 
		компрессора) - позволяет анализировать геометрические параметры лопаток компрессора на 			различных радиусах.
		
	3. Прочностной модуль (доступен только для единичного компрессора) (в перспективе) - 
		позволяет анализировать прочностные свойства лопаток компрессора.
		
		
Процедура пользования сайтом:
	1. Пользователь вводит исходные условия, для расчета компрессора по средней линии тока
		(html-форма).
	2. В случае удачного расчет по средней линии тока пользователя перебрасывает на 
		страницу выбора дальнейшего расчета.
	3. Там пользователь может выбрать одну из нескольких опций:
		3.1. Перейти на страницу с заданием условий для профилирования 
			(доступно как для единичного компрессора, так и для диапазона компрессоров).
		3.2. Перейти на страницу для сравнительного анализа компрессоров в диапазоне 
			(доступно только для диапазона).
		3.3. Перейти на страницу для геометрического анализа (только для единичного компрессора).
